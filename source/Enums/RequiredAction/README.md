# Description

The `RequiredAction` enumeration (enum) defines a number of constants used to
define the action required to ensure the desired state of a DSC Resource can be
actioned.

Current values for `RequiredAction` are:

* `None`
* `Get`
* `New`
* `Set`
* `Remove`
* `Test`
* `Error`
